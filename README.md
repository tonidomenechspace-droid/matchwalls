# Machwalls Obsidian Vault

Repositorio de la boveda de Obsidian de Matchwalls.

## Estructura principal

- `raw/`: fuentes y material bruto.
- `wiki/`: paginas mantenidas de la wiki.
- `AGENTS.md`: instrucciones operativas para Codex y otros asistentes.

## Notas de sincronizacion

Este repo ignora estado local de Obsidian, datos privados de plugins, dependencias y caches generadas. Los archivos ignorados siguen existiendo localmente, pero no se suben a GitHub.
