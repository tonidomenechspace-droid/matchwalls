# Data room

## Summary

A data room is a controlled evidence set used for investor, buyer, or M&A review. In this vault, `raw/ERISMAN/` functions as a local MatchWalls/Erismann data-room source set.

## Key points

- It includes confidential legal, fiscal, financial, operational, advertising, analytics, certification, and presentation material.
- It should not be treated like normal raw marketing content.
- Wiki pages should summarize structure and evidence categories without copying sensitive detail.
