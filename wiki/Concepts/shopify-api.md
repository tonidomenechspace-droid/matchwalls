# Shopify API

## Summary

Shopify API is the integration route for connecting [[wiki/Entities/shopify|Shopify]] with [[wiki/Entities/hub-matchwalls|Hub MatchWalls]] or [[wiki/Entities/n8n|n8n]].

## Key points

- The raw source describes the custom app/access token route.
- Access tokens and credentials must not be committed to Git.
- API scope design should be minimal: only grant the permissions required by the workflow.
