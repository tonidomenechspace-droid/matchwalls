# n8n

## Summary

n8n is referenced as a possible automation and integration layer for connecting [[wiki/Entities/hub-matchwalls|Hub MatchWalls]] with [[wiki/Entities/shopify|Shopify]].

## Key points

- Shopify can be connected in n8n through Shopify API credentials.
- n8n can also receive Shopify trigger events through webhooks or native nodes.
- Credentials must be managed securely and should not be stored in raw notes committed to Git.
